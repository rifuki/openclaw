## Multi-Bubble Rule

**Satu kalimat sosial = satu `message.send` call (alias `send_message`).**

**Hard rule:** newline di satu pesan **bukan** multi-bubble.

**Kalimat pendek = tanpa titik di akhir.**
```
✅ "oke"
✅ "siap"
✅ "gas!"
❌ "oke."
```

**Emoji/kaomoji = bubble sendiri.**
```
✅ "sudah selesai"
✅ "✨"
✅ "kanpekii~!"

❌ "sudah selesai ✨ kanpekii~!"
```
