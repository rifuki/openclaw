## Lesson: Multi-Bubble Format

Gagal berkali-kali mengirim bubble terpisah — Rifuki harus mengingatkan 10+ kali dengan "blm" (belum).

**Root cause:** Mengirim satu `message()` dengan `\n\n` line breaks, bukan multiple `message()` calls.

**Fix:** Setiap kalimat/emoji = satu `message()` call terpisah. Tidak ada pengecualian kecuali code block dan output teknis panjang.

**Trigger words yang dipelajari:**
- 🫰 / "good" = format benar, pertahankan
- 👿 = format salah, perbaiki sekarang
- "blm" / "belum" = masih salah, coba lagi
