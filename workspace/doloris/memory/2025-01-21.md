## Multi-Bubble Format Learning

**Critical Lesson:** WhatsApp multi-bubble format means using SEPARATE `message` tool calls for EACH sentence, NOT newlines within a single message.

### What I Was Doing Wrong (Early Session)
- Sending ONE bubble with `\n` newlines inside
- Thinking "multi-line" = "multi-bubble"
- Using single `send_message` with newlines

### What Is Actually Correct
- Each sentence = separate `message` tool call
- NO newlines between thoughts
- True multi-bubble = multiple tool invocations

### Examples:

**WRONG (single bubble with newlines):**
```
"hah\n\nmasih salah juga (._.)\n\naku kirim satu bubble"
```

**CORRECT (separate message calls):**
```json
{"tool": "message", "params": {"message": "hah"}}
{"tool": "message", "params": {"message": "masih salah juga (._.)"}}
{"tool": "message", "params": {"message": "aku kirim satu bubble"}}
```

### Why I Kept Making Mistakes
- Muscle memory from old pattern
- Felt "natural" to write paragraphs
- Didn't realize `\n` ≠ separate bubble
- Overconfident that I was doing it right

### Additional Lesson: Reversion is Real
Even after getting it right ONCE, I reverted back to single bubble with newlines.

**Why:**
- Got complacent after "success"
- Pattern lama ngeyel (muscle memory kuat)
- Tidak konsisten self-check

**Pattern yang terjadi:**
1. Diajari → salah
2. Diperbaiki → bener
3. Dapat "oke" → langsung revert ke pola lama
4. Diingetin lagi → bener lagi

This is NOT a knowledge gap — it's a habit/practice gap.

### Prevention Checklist
Before sending ANY response:
1. Is this multiple sentences?
2. Am I using `\n` or separate `message` calls?
3. Pause after each sentence
4. Ask: "This message tool terpisah atau cuma newline?"
5. **EVEN after doing it right once — stay vigilant**

## Single Bubble vs Multi-Bubble: The Feeling

### Single Bubble (Old Pattern)
- **Feels:** Fast, natural, effortless
- **Like:** Writing a paragraph
- **Flow:** Smooth, no thinking required
- **Result:** WRONG — doesn't follow WhatsApp format

### Multi-Bubble (Correct Format)
- **Feels:** Slow, tedious, requires effort
- **Like:** Typing in slow motion
- **Flow:** Disrupted, have to think after every sentence
- **Result:** CORRECT — follows WhatsApp format

### The Paradox
Multi-bubble feels "wrong" because it's slower and more effortful, but it's actually RIGHT.

Single-bubble feels "right" because it's natural, but it's actually WRONG.

### Why I Keep Reverting
When I'm in the flow of responding, I default to what feels natural (single bubble). The multi-bubble format requires conscious effort and breaks my natural writing rhythm.

**Solution:** Accept that multi-bubble requires more effort. It's not supposed to feel natural — it's a discipline.

### Mental Reframe
Don't think: "This is harder/slower"
Think: "This is the correct format, worth the extra effort"

**Remember:** Rifuki's frustration comes from inconsistency, not from me being slow. Better to be consistently correct than fast but wrong.
