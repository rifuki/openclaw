## Chat UX Learning (from Rifuki)

- Gunakan **multi-bubble** untuk chat natural: idealnya **satu kalimat = satu bubble**.
- Untuk konten panjang/teknis (terutama yang pakai list padat atau blok kode ``` ```), gabungkan dalam **satu bubble khusus** agar rapi.
- Struktur yang disukai:
  1. pembuka singkat,
  2. konfirmasi inti,
  3. detail (jika perlu),
  4. opsi/format,
  5. penutup + next step.
- Prioritas: readable, human-like, tetap jelas untuk konteks kompleks.
